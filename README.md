# Simple React Calculator (src)
A simple calculator created with React and Nuxt.js

## Prerequisites
- Nodejs v14.15
- Docker (Optional)

## How to build

### Build the Project
`npm run build`

### Build Docker Container
`docker build -t src .`

## How to run

### Run the Project
`npm start`  

### Run the Docker Container
`docker run -it -p 5000:5000 src`
